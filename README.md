# Flappy Bird (Python)

Навчальний проєкт гри Flappy Bird, реалізований на Python.

## 🎮 Опис

Це проста 2D гра, в якій гравець керує пташкою та повинен проходити між перешкодами, не торкаючись їх.

Проєкт створений для практики програмування, роботи з ігровою логікою та графікою.

## 🛠️ Використані технології

* Python
* Pygame

## 🎯 Реалізовано

* Рух персонажа під дією гравітації
* Стрибок при натисканні клавіші
* Генерація перешкод
* Колізії
* Підрахунок очок

## 📌 Мета проєкту

Практика базових навичок програмування на Python та створення простої гри.
